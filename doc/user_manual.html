Manage time

  • Go to project / add project and fill-in the form for each of your projects
  • Go to task / add task and fill-in the form for each of your tasks. Don't forget to associate your tasks with the relevant projects.
  • Go to my time, select a project, the associated tasks will appear in the drop down below ; select the task
  • Add the duration of the task OR the start time and end time. If you fill all, the duration will be overwritten with the computed duration. If you add '+d' after the end time (e.g. 02:30+d) it will break the event in two : one event for the day (up to midnight) and the remaining in the next day.
  • You can add a remark

RSS and Atom feed

RSS and Atom feed are available for any user. Click on the icon.
